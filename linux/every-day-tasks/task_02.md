# our-big-project
Добро пожаловать в проект our-big-project. 

## Описание проекта
Наше приложение реализует простой HTTP-сервер на Python без внешних библиотек.
Сервер слушает на всех интерфейсах (0.0.0.0) порт 8090. Логи записываются в файл /var/logs/big-project-logs/app.log. Если директория не существует, она будет создана.
Для каждого входящего HTTP-запроса логируется IP-адрес клиента и содержимое запроса. В ответ сервер всегда возвращает HTTP 200 с текстом "Запрос получен и залогирован".

Код лежит тут https://gitlab.com/dvps-issues/apps/-/blob/main/big-project.py?ref_type=heads

## Задача
Ваша первая задача - раскатать (запустить) локальную копию бэкенда.
### Шпаргалка по запуску
`python app.py` (запуск лучше с правами, позволяющими записывать в папку для логов).

Для остановки сервера – прервать процесс (`Ctrl+C`).
