# our-big-project
По проекту our-big-project пришла новая задачка - подготовить новый сервер приложения для внешней команды.

Необходимо
- создать новую ВМ (ОС - ununtu/debian)
- установить nginx
- запустить приложение
  - код должен лежать в папке `/var/www/big-project-backend`
  - файл `readme.md` должен лежать в папке `/opt/big-project-docs`
  - логи пишутся по дефолтному пути `/var/log/big-project-logs`
- настроить проксирование запросов на приложение через nginx
