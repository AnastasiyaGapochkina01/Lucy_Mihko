# side-project
DevOps из соседнего проекта ушел в отпуск. Нужно помочь команде запустить новый проект:
1) Создать новую ВМ с Linux на борту
2) На эту ВМ установить golang
3) Создать директорию `/var/www/hello_button`
4) В директорию `/var/www/hello_button` создать файл, который лежит по ссылке: https://gitlab.com/dvps-issues/apps/-/blob/main/hello.go?ref_type=heads;
Это простой веб-сервер, который отображает на ip:port (значение порта - 9115) статическую страничку
5) Запустить приложение одним из способов ниже
  - первый
    ```bash 
       cd /var/www/hello_button
       go run hello.go > /dev/null 2>&1 &
    ```
  - второй
    ```bash
      go build -o hello-server hello.go
      ./hello-server  > /dev/null 2>&1 &
    ```
6) Найти в списке процессов запущенное приложение
7) Найти среди слушающихся портов порт запущенного приложения
8) Для выбранного способа запуска написать unit-file и запустить приложение через systemd
9) Для запущенного в п8 приложения выяснить
- PID
- PPID
- статус (active/exited/failed)
- добавлен ли сервис в автозагрузку
10) Написать инструкцию по разворачиванию и проверке данного приложения
