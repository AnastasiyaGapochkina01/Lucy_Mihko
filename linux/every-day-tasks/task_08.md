# Автоматизация
Для всех наших проектов нужно сделать автоматизацию на bash:
- установка пакетов
  - curl
  - wget
  - make
  - git
  - docker
- создание пользователя `deploy`
- настройка sudo без пароля для пользователя `deploy`
В скрипте надо добавть возможность выбора дополнительных пакетов при запуске
- mariadb
- nginx

Примеры использования скрипта
1. Установка дефолтных пакетов
```bash
./prepare.sh
```
2. Установка дефолтных пакетов И nginx
```bash
./prepare.sh nginx
```
