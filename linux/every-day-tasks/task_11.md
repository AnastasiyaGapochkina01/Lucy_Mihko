# Мониторинг
Для всех проектов нужно реализовать bash-скрипт для простейшего мониторинга
- проверка доступности по http (проект считается недоступным если провалилось три подряд проверки с интревалом 30s)
- проверка статуса systemd-сервиса (или контейнера, если приложение докеризовано)

Результаты должны записываться в файл `bash_montiring.log`
